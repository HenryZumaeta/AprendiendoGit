Comandos Básicos

Los comandos más utilizados en Git para manejar tus repositorios.

  • git init - Crear un nuevo repositorio
  • git clone - Clonar un repositorio existente
  • git add - Añadir archivos al índice
  • git commit - Confirmar cambios
  • git push - Enviar cambios al repositorio remoto
  • git pull - Actualizar el repositorio local desde el remoto

Ramas

Cómo manejar y trabajar con ramas en Git.

  • git branch - Crear y listar ramas
  • git checkout - Cambiar de rama
  • git merge - Fusionar ramas
  • git rebase - Rebase de ramas
